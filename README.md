# Project Title: 5 Day Weather Application

Utilizing the [OpenWeather One Call API](https://openweathermap.org/api/one-call-api) to retrieve weather data for cities, our task was to create a webpage that displayed weather details in a 5-day forecast and a city weather summary. 

VIEW PROJECT - https://maximosandoval.github.io/weather-app/

## Built With

  - HTML, CSS, JS
  - OpenWeather API
